 	Goal
Given a number you must multiply it by itself and all numbers below it and return the total sum of each multiplication.

Example 1:

Input = 5

5*5+5*4+5*3+5*2+5*1=75

return 75

Example 2:

Input = 3

3*3+3*2+3*1=18

return 18

Input


Line 1: An integer N as the number to use to multiply.

Output

Line 1: An integer.

Constraints

0 ≤ N ≤ 9999

Example

Input

4

Output

40
